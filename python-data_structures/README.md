# Python - Data Structures: Lists, Tuples

This project contains Python scripts that demonstrate basic operations on lists and tuples, as required by the ALU Higher Level Programming curriculum.

## Files

| Filename                          | Description                                                                 |
|-----------------------------------|-----------------------------------------------------------------------------|
| `0-print_list_integer.py`         | Prints all integers of a list, one per line.                               |
| `1-element_at.py`                  | Retrieves an element from a list like in C (returns `None` if index invalid). |
| `2-replace_in_list.py`              | Replaces an element at a specific position (modifies the original list).   |
| `3-print_reversed_list_integer.py`  | Prints all integers of a list in reverse order.                            |
| `4-new_in_list.py`                   | Replaces an element in a list without modifying the original list.         |
| `5-no_c.py`                          | Removes all characters `c` and `C` from a string.                          |
| `6-print_matrix_integer.py`          | Prints a matrix (list of lists) of integers.                               |
| `7-add_tuple.py`                      | Adds two tuples element-wise (first two elements only).                    |
| `8-multiple_returns.py`                | Returns the length of a string and its first character.                    |
| `9-max_integer.py`                     | Finds the biggest integer in a list (without using `max`).                 |
| `10-divisible_by_2.py`                  | Returns a list of booleans indicating which integers are divisible by 2.   |
| `11-delete_at.py`                        | Deletes the item at a specific position in a list (without using `pop`).   |
| `12-switch.py`                            | Swaps the values of two variables `a` and `b`.                             |

## Requirements

- All scripts are written in Python 3.8.5 and are executable.
- Code follows `pycodestyle` (version 2.7.*).
- No external modules are imported.
- Functions use `str.format()` where required.
